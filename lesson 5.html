Design Specifications

  • Dimensions: 1080x1080px (Instagram square format)
  • Font: Bold, sans-serif (Helvetica, Arial, or Montserrat)
  • Main Text Size: 48-60px
  • Sub Text Size: 32-40px
  • Text Color: White or high contrast
  • Background: Gradient or solid color

Suggested Color Palette

#667eea
#764ba2
#f093fb
#f5576c
#4facfe

How to Create These in Canva:

  1. Open Canva and select "Instagram Post" template
  2. Choose "Blank" to start from scratch
  3. Add background gradient using the colors above
  4. Add text elements with the fonts and sizes specified
  5. Center align all text
  6. Add subtle drop shadows for better readability
  7. Export as PNG in high quality

Alternative: Use Instagram's Built-in Tools

  1. Take a solid color photo or screenshot of the gradients above
  2. Upload to Instagram Stories
  3. Add text using Instagram's text tool
  4. Save as draft, then add to carousel post
  5. Repeat for all 4 slides

Pro Tips:

  • Keep text large enough to read on mobile screens
  • Use high contrast between text and background
  • Maintain consistent styling across all slides
  • Test readability by viewing on your phone first
  • Consider adding your photo as a small element in corner of Slide 1 & 2
